# 📖 About

**Go Sehat** is an Android-based healthcare application designed to make healthy living easier. Users can monitor their health information, manage daily activities, and access health-related features through a simple and modern interface.

---

# ✨ Features

- 🔐 Secure Login & Registration
- 👤 User Profile
- ❤️ Health Dashboard
- 📊 Health Monitoring
- 🏃 Daily Activity Tracking
- ⚖️ BMI Calculator
- 🍎 Nutrition Information
- 💧 Water Intake Tracker
- 📝 Health Notes
- 📅 Daily Schedule
- 🔔 Health Reminders
- 📈 Progress Statistics
- ☁️ Cloud Synchronization
- 🌙 Modern Material Design UI
